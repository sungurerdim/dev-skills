# Analytics Tool Comparison

| Tool | Privacy | Self-Host | Cost (start) | Best For |
|------|---------|-----------|-------------|----------|
| Plausible | Excellent | Yes | $9/mo cloud | Web, simple |
| PostHog | Good | Yes | Free (self) | Full-stack, features |
| Matomo | Good | Yes | Free (self) | Web, GDPR |
| Mixpanel | Medium | No | Free tier | Mobile, funnels |
| Amplitude | Medium | No | Free tier | Product analytics |
| GA4 | Low | No | Free | Web, if already using |

**Recommendation for solo devs:** Plausible (web) or PostHog self-hosted (full-stack).
